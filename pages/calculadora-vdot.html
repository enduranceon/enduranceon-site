O que é VDOT?

VDOT é um valor numérico desenvolvido pelo Dr. Jack Daniels que representa sua capacidade aeróbica atual. Ele é calculado com base no seu desempenho em corridas recentes e serve como referência para determinar ritmos ideais de treino.

Como usar os ritmos de treino?

Ritmo de Recuperação: Use para corridas leves após treinos intensos ou competições.

Ritmo Fácil: Ideal para corridas longas e treinos de base que desenvolvem resistência aeróbica.

Ritmo de Maratona: Específico para treinos que simulam o ritmo de prova de maratona.

Ritmo de Limiar: Para treinos no limiar anaeróbico, como tiros de 20-40 minutos ou intervalos de 5-10 minutos.

Ritmo de Intervalo: Para intervalos de 3-5 minutos que visam melhorar o VO2max.

Ritmo de Repetição: Para tiros curtos de 30-90 segundos que trabalham velocidade e economia de corrida.
